Aventura de Santis — Guía de publicación (GitHub Pages y Android)
Estructura:
- index.html, style.css, script.js, favicon.ico
- /assets: bg_music.wav, bubble.wav, celebrate.wav, bob.png, patrick.png, gary.png, squidward.png, plankton.png, mrkrabs.png

Publicar en GitHub Pages (PC):
1) Crea tu cuenta en GitHub como "Santis11".
2) New repository -> nombre: Aventura_de_Santis (público).
3) Add file -> Upload files -> sube todos los archivos y la carpeta assets.
4) Settings -> Pages -> Source: Deploy from a branch; Branch: main / root.
5) La URL será: https://Santis11.github.io/Aventura_de_Santis/

Publicar desde Android (Chrome):
1) Entra a github.com -> Inicia sesión.
2) Crea repo "Aventura_de_Santis".
3) En "Code" -> "Add file" -> "Upload files". Sube los archivos.
   Si no puedes subir la carpeta, crea una carpeta "assets" y luego sube dentro los .wav y .png.
4) Ve a "Settings" -> "Pages" y configura como arriba.
5) Abre la URL pública y compártela con Santis.

Notas:
- Reemplaza las imágenes de /assets por las reales si deseas (mismos nombres de archivo).
- Si el audio no se reproduce solo, toca "Comenzar juego" para habilitarlo.
